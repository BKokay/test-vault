[[java]]
The enhanced for loop is a for-of loop and makes iterating through loops more readable.
# Syntax:
```java
for (ElementType element: elements){

}
```

## Example:
```java
String[] fruits = {"Apple", "Banana", "Plum"};
for(String fruit : fruits){
	System.out.println(fruit);
}
```
