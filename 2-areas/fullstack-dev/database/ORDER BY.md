[[ORDER BY]] is how you want a column to be ordered. options are ascending or descending 

##### the syntax: 
```
SELECT 
	select_list
FROM 
	table_name
ORDER BY
	sort_expression1 [ASC | DESC]
	sort_expression2 [ASC | DESC]
```
