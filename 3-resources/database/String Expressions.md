[[String Expressions]] concatenate or manipulate strings using functions like `||` (concatenation) `UPPER()` `LOWER()` and `SUBSTRING()`

##### example
```
SELECT first_name || ' ' || last_name AS full_name 
	FROM fuel_saver.drivers;
```
